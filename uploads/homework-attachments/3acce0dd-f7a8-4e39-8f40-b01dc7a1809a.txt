crtl + d 取消选区
ctrl + delete 换成黑色
ctrl + j 复制图层
ctrl + L 色阶	
shift + F5 填充


1. 证件照替换背景颜色 图像-调整-替换颜色  钢笔工具
2. 画中画  ctrl + t 状态下调整画中画，调整完后按 ctrl shift alt + t
3. 褶皱叠加 先去色（图像-调整-去色 ctrl shift +u） 然后拖入图案，ctrl + t覆盖然后 滤镜-扭曲-置换 ctrl alt +h 然后选择正片叠底 然后调整色阶（1.右下角的圆球里面的色阶，2.ctrl alt shift +e 复刻一个，然后选中新复刻的ctrl+L 调整色阶）
4. 丝带 先用钢笔工具（B）绘制路径 然后 调整画笔，填涂路径，完了后在编辑，定义画笔预设，就可以有这个路径的画笔了， 接着调整画笔，出现丝带的样式，alt吸取颜色就能画出来丝带了
5. 彩虹 渐变里面 - 旧版渐变 有个彩虹的渐变 弄完后 添加蒙版 使用黑白渐变 让红下面的透明变得更加真实
6. 材料贴图 先把贴图保存为一个psd文件，然后在想要贴图的图片上点击滤镜，滤镜库，选择纹理化，导入贴图文件
